Exotérmica

Uma reação exotérmica é aquela em que o sistema libera calor para o ambiente. Em palavras bem simples: a reação “solta” energia para fora.

Calor saindo

O sistema entrega energia térmica para o ambiente.

Produtos menores

Os produtos ficam com menos energia que os reagentes.

ΔH negativo

Como perdeu energia, o resultado da conta fica menor que zero.

Hprodutos < Hreagentes   →   ΔH < 0
