Conceito

Vamos pensar como se fosse uma história: existe uma bolinha no meio. Essa bolinha é o sistema. Tudo que fica em volta dela é o ambiente. O calor é a energia térmica que pode entrar ou sair.

Sistema

É a coisa que estamos olhando.
Exemplo: o gelo, a vela queimando ou uma mistura dentro do copo.

Ambiente

É tudo que está ao redor.
Exemplo: o ar, a mesa, a mão, o copo e a sala.

Calor

É energia térmica em movimento.
Ele pode entrar no sistema ou sair do sistema.

Sistema
químico
Ambiente = fora Sistema = bolinha Calor = energia térmica

Frase bem fácil: “Sistema é o personagem principal. Ambiente é o cenário ao redor. Calor é a energia que passeia entre eles.”
